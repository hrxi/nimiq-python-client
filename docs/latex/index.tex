\begin{quote}
Python implementation of the Nimiq R\+PC client specs. \end{quote}
\hypertarget{index_autotoc_md0}{}\doxysection{Usage}\label{index_autotoc_md0}
Send requests to a Nimiq node with {\ttfamily Nimiq\+Client} object.


\begin{DoxyCode}{0}
\DoxyCodeLine{import nimiqclient}
\DoxyCodeLine{}
\DoxyCodeLine{client = NimiqClient(}
\DoxyCodeLine{    scheme = "http",}
\DoxyCodeLine{    user = "luna",}
\DoxyCodeLine{    password = "moon",}
\DoxyCodeLine{    host = "127.0.0.1",}
\DoxyCodeLine{    port = 8648}
\DoxyCodeLine{)}
\end{DoxyCode}


Once the client have been set up, we can call the methods with the appropiate arguments to make requests to the Nimiq node.

When no {\ttfamily config} object is passed in the initialization it will use the default values in the Nimiq node.


\begin{DoxyCode}{0}
\DoxyCodeLine{client = NimiqClient()}
\DoxyCodeLine{}
\DoxyCodeLine{\# make rpc call to get the block number}
\DoxyCodeLine{blockNumber = client.blockNumber()}
\DoxyCodeLine{}
\DoxyCodeLine{print(blockNumber) \# displays the block number, for example 748883}
\end{DoxyCode}
